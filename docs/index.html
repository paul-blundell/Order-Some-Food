Order Some Food

Version 2.9
A full ready to use website that mimics many of the features of popular takeaway ordering websites, such as Just-Eat.
This web application will provide you with a ready to use website where takeaway owners will be able to register their establishment, add their menu and start receiving orders online. If you are looking for a script, web application or even a full ready to use website that allows online takeaway ordering then look no further!
  • User friendly interface
  • Search for takeaways in local area
  • Receive orders online
  • Support for multiple languages
  • Support for multiple currencies
  • Accept online payments
  • Easy to customise
  • Full source code available

About

The Order Some Food web application built using PHP and CodeIgniter will provide you with extensive online takeaway ordering business. This will allow owners of takeaway establishments to register their details and begin selling online.

Designed to be as easy to use as possible and even easier to modify is what makes Order Some Food the perfect application for your needs, whether that is establishing a website similar to Just-Eat or Hungry House or perhaps you are business with a chain of restuarants.

Order Some Food can be used for a wide variety of requirements, take a look at our demo now to see if our online ordering application is for you.

Installation

Installation of Order Some Food could not be any easier. Simply extract the zip file into your websites directory, import the provided .sql file into your database and then modify a couple of settings in a configuration file. You're done!

These instructions are provided in more detail in a readme file inside the applications package.
